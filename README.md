y opciones para jugar contra la CPU u otro 
usuario en el mismo sistema.

-Github Pages

## Instalación

1. Clona este repositorio:

```bash
git clone https://github.com/tu-usuario/tic-tac-toe.git
cd tic-tac-toe
```

2. Instalar las dependencias

```bash
npm install
```

3. Inicia la aplicación:

```bash
npm start
```
